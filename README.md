# AI CV Critiquer 📄🤖

A Streamlit web app that uses OpenAI to analyse and critique CVs uploaded in PDF or TXT format.

## 🚀 Features
- Upload CV (PDF or TXT)
- Extract text automatically
- AI-powered CV critique using OpenAI GPT model
- Feedback on:
  - Content clarity
  - Skills relevance
  - Experience quality
  - Job-specific improvements

## 🛠️ Tech Stack
- Python
- Streamlit
- OpenAI API
- PyPDF2
- dotenv

## 📦 Installation

```bash
git clone https://github.com/your-username/ai-cv-critiquer.git
cd ai-cv-critiquer
python -m venv .venv
.venv\Scripts\activate   # Windows
pip install -r requirements.txt
